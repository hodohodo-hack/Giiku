# 👾 Gi育 (Giiku)

**"Grow your spirit, grow your pet."**

Gi育は、日々の開発（Git操作）をキャラクターの育成体験に変換するCLIツールです。  
特定のレポジトリに縛られず、あなたのPC上での全ての活動がキャラクターの糧になります。

---

## ✨ 特徴

- 🥗 **Git操作で育成**: `git commit` でご飯をあげ、`git push` で自分を磨く。
- 🏆 **称号システム**: あなたの開発スタイル（早起き、深夜型、慎重派など）に応じた称号を自動獲得。
- 🎭 **スキン収集**: 開発が進むと新しい種族（スライム、メカ、ゴーストなど）がアンロック。
- 🌏 **多言語対応**: 日本語と英語をフルサポート。
- 🛠️ **モダンな技術スタック**: TypeScript + React (Ink) による、美しく堅牢なCLI。

---

## 🚀 クイックスタート

### 1. インストール
```bash
git clone https://github.com/hodohodo-hack/Giiku.git
cd Giiku
npm install
```

### 2. ビルド & リンク
```bash
npm run build
npm run link-cli
```

### 3. セットアップ
普段お使いのシェル（Bash / Zsh）にGi育を統合します。
```bash
# 日本語でセットアップ
giiku --setup --ja

# 英語でセットアップ
giiku --setup
```
※完了後、ターミナルを再起動するか `source ~/.zshrc`（または `~/.bashrc`）を実行してください。

---

## 🎮 遊び方

- **ターミナルを開く**: キャラクターが現在のコンディションとともに挨拶します。
- **Gitを使う**: `git status` などの後にステータスバーが表示され、`git commit` するとリッチな演出が発生します。
- **管理する (`giiku`)**: 引数なしでコマンドを実行するとTUIモードが起動。スキンの切り替えや詳細な統計確認ができます。
