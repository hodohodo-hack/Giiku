# Gi育 (Giiku) - Git-based Character Growth Tool

## 概要
「Gi育（ぎいく）」は、日々の開発（Git操作）を「キャラクターの育成」に変換し、エンジニアのモチベーションを維持するためのCLIツールです。特定のプロジェクトに依存せず、あなたのパソコン上（グローバル）でキャラクターが成長し続けます。

---

## クイックスタート

### インストール
1.  依存ライブラリのインストール: `npm install`
2.  CLIのビルドとグローバルリンク: `npm run link-cli`
3.  シェルのセットアップ: `giiku --setup` (日本語化したい場合は `giiku --setup --ja`)
4.  ターミナルの再起動、または設定の再読み込み（例: `source ~/.zshrc`）を実行してください。

---

## 育成システム

### コンディション管理
Git操作を「エサ」や「手入れ」に見立て、2つのパラメータで状態を表現します。

- **満腹度 (Satiety)**:
  - `git commit` を行うことで大幅に回復します。
  - **減少量**: 1時間ごとに 2% 減少します。
  - **飢餓状態**: 30%以下になるとキャラクターの見た目や挨拶が変化します。
- **ツヤ (Luster)**:
  - `git push` を行うことで上昇します。
  - **減少量**: 1時間ごとに 1% 減少します。
  - **視覚効果**: 数値が高まると（40%以上）、キャラクターの周囲にオーラ（輝く粒子）が発生します。

### 統計と記録
- **活動日数**: Gi育を導入してから、Git操作を行った通算日数を記録します。
- **コミット数**: PC全体での「合計コミット数」と「今日のコミット数」を表示します。

---

## 称号システム (Identity)
あなたの開発スタイルに応じて、称号が自動的に付与されます。

- **実績称号（永続）**:
  - `早起きのエンジニア🌅`: 朝（5:00-9:00）にコミットを行う。
  - `デバッグの鬼👹`: コミットメッセージに "fix" を多く含む。
  - `百戦錬磨⚔️`: 通算コミット数が100を超える。
- **状態称号（動的）**:
  - `現在：慎重派🐢`: `git diff` や `git status` をコミットの数倍多く実行している。
  - `現在：炎上中🔥`: 深夜（22:00-3:00）に連続してコミットを行う。

## ユーザー体験

### 1. ターミナル起動時の挨拶 (`--startup`)
新しいターミナルを開くたびに、キャラクターが現在のコンディションとともに挨拶します。

### 2. コマンド後のフィードバック
- **1行ステータス**: `git status` や `git push` の後に、現在のパラメータを1行で表示します。
- **リッチ・リアクション**: `git commit` 成功時に、キャラクターが大きく表示され、パラメータの回復を祝います。

### 3. TUI (マネジメントモード)
引数なしで `giiku` を実行すると、フル画面モードが起動します。
- **スキンの切り替え**: **左右の矢印キー**で、4種類の種族（SLIME, MECHA, PHANTOM, FOREST）を切り替えることができます。
- **詳細確認**: 現在の全てのステータスと、獲得している称号を確認できます。

## コマンドリファレンス

| コマンド | 説明 |
| --- | --- |
| `giiku` | TUI (マネジメントモード) を起動 |
| `giiku --help` | ヘルプを表示 (英語/日本語) |
| `giiku --setup` | シェル (.bashrc/.zshrc) への統合を開始 |
| `giiku --setup --ja` | 日本語モードでセットアップを実行 |
| `giiku --remove-setup` | シェルのフックを削除 (アンインストール) |
| `giiku --version` | バージョン情報を表示 |

### デバッグ用オプション (要 `--help --debug`)
| コマンド | 説明 |
| --- | --- |
| `giiku --hook commit` | コミットを擬似実行 (満腹度UP) |
| `giiku --hook push` | プッシュを擬似実行 (ツヤUP) |
| `giiku --hook starve` | 飢餓を擬似実行 (満腹度DOWN) |
| `giiku --hook rust` | 劣化を擬似実行 (ツヤDOWN) |

---

## 技術的な詳細
- **データ保存**: ユーザーのホームディレクトリ（`~/.config/giiku-tui/` など）に保存されます。
- **依存性注入 (DI)**: 描画ロジックとコアエンジンが分離されており、将来的にキャラクター描画エンジンを容易に差し替えることが可能です。
